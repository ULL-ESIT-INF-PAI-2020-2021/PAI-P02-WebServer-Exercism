## Instructions

### Setup

* 1 - Clone this repo in your directory:

````
git clone git@github.com:PAI-ULL/node-express-static-web-server.git <yourDirectory>
````

* 2 - move to yourDirectory:

````
cd <yourDirectory>
````

* 3 - Run this command:

````
npm install
````

* 4 Run this command:

````
npm start
````

* 4 Point your browser to:

````
http://<your VM IP address>:8080
````
